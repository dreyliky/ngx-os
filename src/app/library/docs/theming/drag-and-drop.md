## DraggableDirective

### CSS Classes

Applies for `movable element`. Movable element MIGHT BE child of element[osDraggable].

| Name            | Description                       |
| --------------- | --------------------------------- |
| `.os-draggable` | Draggable element                 |
| `.os-movable`   | Movable element                   |
| `.os-dragging`  | Is element dragging right now?    |
