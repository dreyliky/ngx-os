## Description

Allows organizing content across different screens, data sets, and other interactions.

```html
<os-tab-group>
    <os-tab
        [label]="'Tab #1'">
        Content of the tab #1
    </os-tab>

    <os-tab
        [label]="'Tab #2'">
        Content of the tab #2
    </os-tab>
</os-tab-group>
```
